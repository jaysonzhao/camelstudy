Terminology

Integration - A set of ordered steps that Red Hat Fuse Online executes.  Each step operates on data.

  • Start: A step that connects to an application to start the integration. Provides initial data.

  • Optional additional steps that connect to applications between the start and finish connections.

  • Finish: A step that connects to an application to complete the integration. Sends final data of all previous steps to output application.

Connectors - Represent specific applications which send or receive data.

  • Template for connecting to an external application.

  • Set of connectors provided by Red Hat Fuse Online.

  • Custom connectors to other applications can be created.

 

Connections - A connector with configuration information.

  • A connection is a specific instance of a connector.

  • Multiple connections for the same connector can be created.

 

Actions - Each connection performs exactly one action in an integration.

Data Mapper

  • Lets you map shaped output from the previous steps to the shaped input of the next step.

  • All output from previous steps is available for mapping.

  • Supports:

    • Splitting fields

    • Combining fields

    • Field level transformations

Migration

  • Export of integration includes everything you need to run that integration (connections, extensions, etc.) in a single ZIP file

  • Import adds an integration and all dependencies into a Red Hat Fuse Online instance

  • Connection may need to be updated when moving between environments

  • Environment portability includes cloud and on-premise
